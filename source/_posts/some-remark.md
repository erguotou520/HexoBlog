* 登录到ssh
```
ssh git@127.0.0.1
```

* 查看版本和远程url
```
git remote -v
```

* git迁移  
1.clone一份或者直接在原本地工程上开始
2.`git remote set-url origin <url>` url为目标迁移的git地址
3.`git push origin master` 将本地资源push到远程
